## 🚀 Introduction
Astute Graphics for Windows is a set of plugins that install into Adobe Illustrator and rethink the slower parts of vector work. Rather than manually correcting anchor points or redrawing forms, you work with live dynamic shapes, curve handles that respond intelligently, and cleanup tools that simplify paths in a click. The complete Astute Graphics illustrator toolkit is packaged together as the Elite Bundle, so the drawing, editing, color, and productivity plugins all arrive in one install and stay ready inside Illustrator.

## ✨ Highlights

### Drawing and Editing
- **Dynamic Shapes** — Build rectangles, ellipses, and custom forms that stay live and editable long after you draw them.
- **Smart Handles** — Adjust Bézier curves with handles engineered to make anchors and tangents easier to control.
- **Precise Vector Cleanup** — Strip redundant points, close gaps, and tidy artwork without changing its shape.

### Color and Productivity
- **Color and Texturing Tools** — Sample, blend, and manage color in dedicated panels, and add textures to flat vector art.
- **Elite Bundle** — Get the full Astute Graphics plugin collection together for a complete Illustrator productivity upgrade.

**Which versions of Windows are supported?**
